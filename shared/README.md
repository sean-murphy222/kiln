# Shared

Shared utilities and common code across all Kiln modules.

## Contents

- **Data models:** Common data structures used across modules
- **Export formats:** Standardized JSON/JSONL schemas
- **Utilities:** Logging, configuration, file handling
- **Type definitions:** Shared TypeScript/Python types
- **Constants:** Project-wide constants and enums

## Usage

All modules (Quarry, Forge, Foundry, Hearth) can import from `shared` to ensure consistency in data formats and reduce code duplication.
